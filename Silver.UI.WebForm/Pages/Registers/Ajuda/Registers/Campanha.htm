Campanha

Na tela de campanha definimos as estrategias e configurações das campanhas que relacionaremos com os usuarios.

Um grupo de usuario pode ter varias campanhas sendo um por usuario.

Nesta tela temos os seguintes botões:

Cadastro de uma nova campanha.
Pesquisar campanha existe.
Altera dados da campanha selecionada.
Exclui a campanha selecionada.


Clicando no botão de Novo ou Alteração uma tela sobreposta com os seguintes campos:

Nome Nome da Campanha sem Espaço ou Acentos.
Descrição Campo livre para descrição da Campanha.
Música de espera Musica que o Cliente escutará ao atender o telefone originado pelo discador, até a comutação com o Atendente.
Estratégia Estrategia de tribuição das chamadas.

Temos os seguintes opções:

Tocar todos os agentes simultameamente: O sistema irá tocar para os os Atendentes e irá transferir para o primeiro que tirar do gancho.
Tocar os agentes randomicamente: O sistema irá excolher aleatoriamente o proximo Atendente que receberá a ligação.
Tocar os agentes na ordem do ramal: O agente irá transferir a ligação na ordem do Ramal.
Tocar o agente com o maior tempo sem ligação: O sistema irá tranferir para o Atendente maior tempo de espera.
Tocar o agente com menos ligação: O sistema irá transferir para o agente que menos recebeu ligação.
Tocar os agentes um a um, sequencialmente: O sistema irá transferir na ordem, sem se importar com o tempo de espera do atendente.
Quantidade de Toques Quantidade de toques que tentaremos fazer ao cliente para esperar o atendimento.
Tempo de Espera Tempo de espera entre uma transferencia e outra para registro de Historico.
Anuncio Mensagem que será emitido ao cliente de tempo em tempo como anuncio, exemplo: "Aguarde, será atendido logo".
Pausa Os tipos de pausas que o usuario poderá usufluir quando estiver nesta campanha.
Ativo Flag para definir se a campanha esta ativa ou em Standby.
Salvar os dados informados.
Cancela as alterações na campanha.
